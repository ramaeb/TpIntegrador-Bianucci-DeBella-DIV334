Proyecto Integrador Ramiro Bianucci y Nicolas De Bella.

El proyecto consiste en una aplicacion web, la cual tiene un carro de compras para el usuario de productos, en este caso utiliza las LICENCIAS de WINDOWS y varias licencias más.
El usuario puede poner todo en un carrito y terminar la compra con un TICKET el cual guarda el valor a pagar e imprime los productos elegidos para su posterior pago.

La aplicacion contiene un apartado de ADMIN, el cual luego de loguearse contiene un dashboard que tiene la opcion de ELIMINAR MODIFICAR VER Y CONSULTAR  la cual permite al administrador tener una vista y controlar los productos, modificar su precio y más.
